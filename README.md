# Hotel-Management

## Installation
1. Install XAMPP.
2. Open XAMPP Control Panel and start Apache and Mysql. Make sure they both are started.
3. Download the code.
4. Extract to C:\xampp\htdocs.
6. Open any browser and type http://localhost/Hotel-Management.
7. Now enjoy the code.:fire:
